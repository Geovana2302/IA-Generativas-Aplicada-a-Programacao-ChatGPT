# 🤖 IA Generativa Aplicada à Programação - ChatGPT

Repositório criado para armazenar os exercícios e atividades desenvolvidos
durante o **Curso de Aperfeiçoamento Profissional de Inteligências Artificiais
Generativas Aplicadas à Programação com ChatGPT**.

## 📖 Sobre o Curso

O curso tem por objetivo a implementação de soluções de desenvolvimento por
meio de inteligência artificial generativa, abordando:

- Identificação e correção de erros comuns em scripts
- Reescrita de códigos para melhorar legibilidade e desempenho
- Aprimoramento do aprendizado de novas tecnologias, como linguagens de
  programação, bibliotecas, frameworks e documentações
- Treinamento personalizado do ChatGPT via API da OpenAI
- Criação e disponibilização de chatbots de prompt de texto para
  diversas necessidades de mercado

## 📚 Conteúdo do Repositório

- Exercícios práticos realizados ao longo do curso
- Atividades e desafios propostos
- Experimentos com prompts e integração com a API da OpenAI
- Projetos de chatbot desenvolvidos durante o curso

## 🛠️ Tecnologias

- ChatGPT / OpenAI API
- Python
- Bibliotecas e frameworks abordados no curso

## 🎯 Objetivo

Documentar o aprendizado e evoluir na utilização de IAs generativas
como ferramentas no dia a dia da programação, desenvolvendo soluções
práticas e aplicáveis ao mercado.
